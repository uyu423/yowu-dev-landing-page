# Yowu.dev Landing Page

Backend Developer Yowu의 개인 포트폴리오 랜딩 페이지입니다.

## Project Overview
- **Design**: Modern Dark Theme, Video Background, Glassmorphism UI.
- **Tech Stack**: HTML5, CSS3, Vanilla JS (No External Frameworks).
- **Hosting**: GitHub Pages.

## Features
- **Hero Section**: 프로필, 소개, 통합 링크(Resume, GitHub, Wiki 등).
- **About Section**: 경력 및 주요 통계 요약.
- **Responsive**: 모바일 및 데스크탑 대응 (Flexbox Layout).

## Development
- **Branch Strategy**: `master` (Production), `feature/*` (Development).
- **Convention**: HTML/JS (Tab), CSS (2 Spaces).

## History
- **Refactor**: jQuery 및 Legacy 라이브러리(skel.js 등) 제거.
- **Redesign**: 기존 템플릿(Visualize)에서 Modern Dark Theme로 전면 리뉴얼.
